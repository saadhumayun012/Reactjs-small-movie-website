## ReactJS Small Movie Website 

This is a small movie website built using **ReactJS**.
In this project, users can see movies and view their details using real movie data.

The movie data is fetched from **TMDB (The Movie Database) API**.

This project is mainly created for **learning React**, components, context API, and API integration.

---

## Features

* Show movies on home page
* Movie details page
* Real data fetched from TMDB API
* Built using React + Vite

## TMDB API (Important)

This project **will not run without a TMDB API key**.

To run this project:

1. Create an account on **[https://www.themoviedb.org/](https://www.themoviedb.org/)**
2. Generate your API key
3. Add the API key and base url in your `.env` file

Example:

```
VITE_TMDB_API_KEY = ""
VITE_TMDB_BASE_URL = ""
```

---

## How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/saadhumayun012/Reactjs-small-movie-website.git
```

2. Go to project folder:

```bash
cd Reactjs-small-movie-website
```

3. Install dependencies:

```bash
npm install
```

4. Start the project:

```bash
npm run dev
```
